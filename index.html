Utility

Fixed-Width Output Generator

Define the columns of a fixed-width record — length and type — and the tool randomly generates output lines that fit the spec exactly.

01 Field Definitions

Number, Uppercase, Lowercase, and Modifiers fill the full length with random digits, letters, or symbol characters. Space is always blank filler.

Name Length Type Align Pad

Paste a 2-column list (Length, Type) — names are auto-generated — or a 3+ column list (Name, Length, Type, [Align], [Pad]). Tab-separated (Excel paste) or comma-separated both work. A header row is detected and skipped automatically. Valid types: Number, Uppercase, Lowercase, Modifiers, Space.

02 Output

Every line is exactly 0 characters wide, randomly generated from the field definitions above.
