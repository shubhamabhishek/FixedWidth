Utility

Fixed-Width Output Generator

Define the columns of a fixed-width record — length and type — and the tool randomly generates output lines that fit the spec exactly.

01 Field Definitions

Number, Uppercase, Lowercase, and Modifiers fill the full length with random digits, letters, or symbol characters. Constant always outputs whatever you type in the Value column (length is locked to match). Date outputs the picked date as YYYYMMDD. Space is always blank filler. Drag the handle on the left to reorder fields.

Name Length Type Value

Paste a 2-column list (Length, Type) — names are auto-generated — or a 3-column list (Name, Length, Type). Tab-separated (Excel paste) or comma-separated both work. A header row is detected and skipped automatically. Valid types: Number, Uppercase, Lowercase, Modifiers, Space, Constant, Date. For Constant, set the output value afterward in the Value column — length auto-adjusts to match. For Date, the date defaults to today and can be changed afterward in the table.

02 Output

Every line is exactly 0 characters wide, randomly generated from the field definitions above.
